# CySA+ Study Notes

This repository contains my structured notes while studying for the CompTIA CySA+ certification.

## Topics Covered

- Threat and vulnerability management
- Security operations and monitoring
- Incident response
- Reporting and communication
- Security tools and analysis concepts

## Purpose

The goal of this repository is to organize key CySA+ concepts in a clear format that supports exam preparation and reinforces practical SOC analyst knowledge.

## Repository Structure

- `threat-and-vulnerability-management` - notes on identifying and managing risk
- `security-operations-and-monitoring` - notes on logs, alerts, SIEMs, and detection
- `incident-response` - notes on investigation and response processes
- `reporting-and-communication` - notes on documentation and stakeholder communication
- `tools-and-queries` - notes on security tools, queries, and technical examples
